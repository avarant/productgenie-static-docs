Welcome to the ProductGenie API documentation.

Our API allows you to integrate ProductGenie features directly into your applications and workflows. You can manage products, handle reviews, get product suggestions, process customer questions, and more.

## Getting Started

1.  **Authentication:** Most endpoints require authentication. Refer to the specific endpoint documentation for details on required headers (`X-Public-Key`, `X-Productgenie-Signature`).
2.  **Integration:** Learn how to embed the ProductGenie widget on your website using our [Integration Guide](./guides/integration-guide/).
3.  **Base URL:** All API endpoints are relative to `https://api.productgenie.io`.

## Available APIs

Use the sidebar navigation to explore the different API categories:

-   [Contact](./reference/contact/): Submit contact form messages with reCAPTCHA validation
-   [Message](./reference/message/): Manage conversations with the AI assistant
-   [Products](./reference/products/): Create, update, delete, and retrieve product information
-   [Reviews](./reference/reviews/): Manage product reviews
-   [Suggestions](./reference/suggestions/): Get AI-generated suggested questions for products

## API Overview

### Public Endpoints
- **Contact API**: Submit inquiries and support requests
- **Message API**: Interactive conversations with AI assistant
- **Suggestions API**: Retrieve product-specific questions

### Authenticated Endpoints
- **Products API**: Full CRUD operations for product management
- **Reviews API**: Create, update, and delete customer reviews

### Authentication Methods
- **Read Operations**: Require only `X-Public-Key` header
- **Write Operations**: Require both `X-Public-Key` and `X-Productgenie-Signature` headers
